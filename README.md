🌦 Live Weather Analytics Dashboard | Power BI Project

This project showcases a dynamic and interactive weather analytics dashboard built using Power BI. The dashboard transforms real-time weather data into meaningful insights using data modeling and DAX measures.

📊 Dashboard Features

🌡 Real-time Temperature Monitoring

📈 7-Day Temperature Trend

🌅 Sunrise & Sunset Tracking

🌍 Air Quality Index (AQI) Gauge

💨 Wind Speed, Humidity & Visibility Metrics

🌧 Rain Probability Analysis
